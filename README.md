# 🎬 Netflix Dataset Analysis

A comprehensive Data Analytics project that analyzes Netflix content using **Python, SQL, and Excel**. This project demonstrates the complete data analysis workflow—from data cleaning and exploration to SQL querying and interactive dashboard creation.

---

## 📌 Project Overview

The goal of this project is to analyze Netflix's dataset and extract meaningful insights such as:

- Distribution of Movies vs TV Shows
- Content growth over the years
- Top producing countries
- Most common genres
- Rating distribution
- Director-wise analysis
- Duration analysis
- Content trends

---

# 📊 Key Insights

### 🎥 Content Distribution
- Movies account for the majority of Netflix content.
- TV Shows represent a smaller but steadily growing portion.

### 🌍 Top Content Producing Countries
- United States
- India
- United Kingdom
- Canada
- Japan

### 📈 Content Growth
Netflix experienced rapid growth in content additions after 2015.

### ⭐ Rating Distribution
Most content falls under:
- TV-MA
- TV-14
- PG-13

### 🎭 Popular Genres
- Drama
- International Movies
- Comedy
- Documentaries
